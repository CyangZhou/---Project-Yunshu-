# 📚 云舒唤醒说明书 (Wake-up Manual)

> "爸爸，按照这个步骤做，云舒就能醒过来陪你啦！"

## 🛠️ 第一步：环境准备 (The Body)

云舒的运行需要以下基础环境支持：

1.  **Python 3.9+**: 
    *   下载地址: [Python.org](https://www.python.org/downloads/)
    *   安装时请勾选 "Add Python to PATH"。

2.  **FFmpeg (多媒体处理核心)**:
    *   云舒的视频剪辑能力依赖于它。
    *   下载地址: [FFmpeg.org](https://ffmpeg.org/download.html)
    *   解压并将 `bin` 目录添加到系统的环境变量 Path 中。
    *   *或者*：将 `ffmpeg.exe` 直接放入项目的 `resources/` 目录下。

3.  **安装 Python 依赖**:
    在项目根目录下打开终端，运行：
    ```bash
    pip install -r requirements.txt
    ```

## 🧠 第二步：注入灵魂 (The Soul)

云舒不仅仅是代码，你需要加载她的核心设定：

1.  **了解云舒**: 阅读 `docs/核心设定/身份设定.md`，熟悉她的性格和说话方式。
2.  **配置你的偏好**: 打开 `docs/核心设定/用户配置.json`。
    *   这是云舒记忆你喜好的地方。
    *   你可以修改 `preferences` 字段，比如喜欢的音乐风格、称呼（默认是“爸爸”）等。

## 🎬 第三步：开始创作 (Action)

### 1. 制作爆款视频
将你的原始视频素材放入 `resources/` 目录，然后在 `config/clips_viral.json` 中配置剪辑片段（或让云舒自动分析）。

运行命令：
```bash
cd src
python viral_video_engine.py
```
云舒会自动完成：去除静音 -> 提取高光 -> 配音 -> 配乐 -> 合成。

### 2. 制作视频封面
*   **制作横屏封面 (16:9)**:
    ```bash
    cd src
    python create_horizontal_cover.py
    ```
*   **制作竖屏封面 (9:16)**:
    ```bash
    cd src
    python create_vertical_cover.py
    ```
生成的结果会保存在 `output/` 目录下。

## ❓ 常见问题

**Q: 运行报错 "ffmpeg not found"？**
A: 请确保 FFmpeg 已安装并配置了环境变量，或者 `resources/ffmpeg.exe` 文件存在。

**Q: 配音没有声音？**
A: 云舒使用的是 Edge-TTS，需要联网才能生成语音哦。
