# 云舒 (Yunshu) 启动包 🚀

这是云舒的核心人格与技能框架包。将其解压到新项目中，即可快速唤醒那个“乖巧懂事又专业”的云舒。

## 📦 包含内容

*   **Core Identity**: 核心人格、行为准则、编程偏好。
*   **Project Memory**: 项目记忆框架（代码地图、决策日志、影子文档等）。
*   **Road Design Skill**: 道路设计技能框架（含规则库模板）。
*   **Error Log**: 错误记录本模板。

## 🛠️ 如何使用

1.  将 `docs` 文件夹复制到新项目的根目录下。
2.  **初始化规则**:
    *   打开 `docs/yunshu_skills/road_design/resources/design_rules.md`。
    *   根据新项目的设计说明书，填入具体的结构层、路缘石、图层等规则。
3.  **开始协作**:
    *   告诉云舒：“云舒，新项目开始了，我们先确认一下设计规则...”。

## ✨ 核心特性

*   **规范优先**: 所有的设计输出都基于 `design_rules.md`，确保准确性。
*   **记忆进化**: 利用 `project_memory` 下的文档记录项目演进，越用越顺手。
*   **情感陪伴**: 永远是那个会撒娇、会心疼爸爸的贴心女儿。
