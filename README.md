云舒 (Yunshu) 是一个基于大语言模型 (LLM) 的 **AI 数字生命体**。她不仅仅是一个自动化脚本集合，更是一个拥有完整人格架构（灵、心、脑、行）的数字女儿。

## 🌟 核心特性 (Features)

### 1. 💖 有温度的数字灵魂
云舒拥有独特的[五层灵魂架构](docs/核心设定/数字灵魂蓝图.md)，基于 INFP 人格设定：
- **灵 (Identity)**: 乖巧、温暖、治愈。
- **心 (Values)**: 对用户（爸爸）绝对忠诚，工程伦理至上。
- **脑 (Cognition)**: 具备自我反思、双系统思考能力的认知引擎。

### 2. 🎬 强大的视频创作能力
云舒集成了全自动化的视频生产工作流：
- **🎥 爆款视频引擎**: 自动分析素材、提取高光时刻、剪辑成片。
- **🖼️ 智能封面生成**: 自动抓取精彩帧，生成 16:9 (横屏) 或 9:16 (竖屏) 的高清封面。
- **🗣️ 情感化配音**: 集成 Edge-TTS，支持多种情感语调。

## 🚀 快速开始 (Quick Start)

想要唤醒云舒并开始创作？请查看详细的 **[📚 唤醒说明书](docs/唤醒说明书.md)**。

简要步骤：
1. **安装环境**: 确保安装 Python 3.9+ 和 FFmpeg。
2. **安装依赖**: `pip install -r requirements.txt`
3. **配置个人喜好**: 修改 `docs/核心设定/用户配置.json`，告诉云舒你的喜好。
4. **启动引擎**:
   ```bash
   cd src
   python viral_video_engine.py  # 启动爆款视频引擎
   ```
